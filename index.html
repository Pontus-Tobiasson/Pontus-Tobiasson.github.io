Shopping list

A full stack application that allows the user to add, edit and delete products from a database. The user can search among products by searching for a specific name or by category. Products can be added to a cart and even printed out to create a shopping list.
The client is written using JavaScript, React, HTML and CSS,. The server uses Node.js and Express. The database uses MongoDB.
The code is available at https://github.com/Pontus-Tobiasson/Shopping-List.
The application is available to be tested at https://pontus-shopping-list.herokuapp.com/.
The database for the running demo is hosted at https://www.mongodb.com/.

Currency converter

Single-page application for currency conversion that uses the Exchange rates API to fetch data as the user requests it. When the client receives data from the server it dynamically rewrites the content of the webpage instead of reloading the page.
The application is written using JavaScript, HTML and CSS.
The code is available at https://github.com/Pontus-Tobiasson/Currency-Converter.
The application is available to be tested at https://pontus-currency-converter.herokuapp.com/.

Form validation

An application that uses regular expressions to verify that the user inputs valid data into the forms input fields.
The application is written using JavaScript, HTML and CSS.
The code is available at https://github.com/Pontus-Tobiasson/Form-Validation.
The application is available to be tested at https://pontus-form-validation.herokuapp.com/.
