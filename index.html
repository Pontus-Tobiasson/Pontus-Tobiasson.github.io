Shopping list

The Shopping List is an application that creates shopping lists. The user can search for groceries by name and category, and select the number of units to purchase. The resulting list can be printed. Groceries can be created, updated and removed.

The application is responsive. The wide view shows the categories, groceries and shopping list side-by-side. In the mid-sized view the shopping list slides in and out on demand. In the narrow view the category list becomes a dropdown list.

The Shopping List is a full stack application built on several technologies:

  • The grocercy data is stored in MongoDB, with full CRUD access
  • The server uses:
    • Node.js
    • Express
    • Sharp to convert dropped images to png thumbnails
    • Formidable to simplify form data processing
  • The client uses:
    • React
    • HTML5 drag and drop to update grocery thumbnails
    • HTML5 canvas to preview dropped images before saving them in the database

Code:
https://github.com/Pontus-Tobiasson/Shopping-List
Demo:
https://pontus-shopping-list.herokuapp.com/

The demo database is hosted at https://www.mongodb.com/.

Currency converter

The Currency converter is a single-page application that shows currency exchange rates. The rates are retrived from api.exchangeratesapi.io, published by the European Central Bank

The application is built exclusively with JavaScript, HTML and CSS. No external libraries or frameworks has been used.
